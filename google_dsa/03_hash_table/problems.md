> 每題限時 **20 分鐘**，超時就看解答、隔天重做。用 Google Doc 或紙筆手寫。

## 代表題（先做這題把模板練熟）
- **Two Sum**（LeetCode 1）
  - 就是本主題模板本體，注意回傳的是**索引**，而且陣列**未排序**。

## 練習清單（第一梯：雜湊表）
| 題目 | 難度 | 重點 |
|------|------|------|
| Two Sum | Easy | 本模板本體，值 -> index 反查另一半 |
| Contains Duplicate | Easy | 用 set 判斷是否出現過 |
| Valid Anagram | Easy | 統計字母出現次數比對 |
| Group Anagrams | Medium | 以排序後字串當 key 分組 |
| Longest Consecutive Sequence | Medium | 用 set 找連續序列起點，O(n) |

## 20 分鐘限時提示
1. **~2 分**：先問清楚——有負數嗎？重複值？空陣列？回傳索引還是值？
2. **~8 分**：先講暴力 O(n²)，再說「用 hash table 記看過的值 → 降到 O(n)、空間 O(n)」。
3. **~5 分**：邊寫邊講，記得先查再寫入（處理重複值）。
4. **~5 分**：拿 `[3, 2, 4], target=6` dry run，逐步講 seen 的變化。
